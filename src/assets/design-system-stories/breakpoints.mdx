# Breakpoints

Breakpoints are used for mobile-first responsive design. As CSS variables can't be used in media queries, we use SCSS variables.

<DesignTokenDocBlock categoryName="Breakpoints" showSearch={false} />

## Usage

Use breakpoints in the custom media query mixin:

```css
@use "@/assets/scss/breakpoints";

@include above(breakpoints.$breakpoint) {
   ...
}
```
